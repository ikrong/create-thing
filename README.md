# create-thing

从空项目开始创建

# cli

本项目注册两个命令，help和thing命令

使用方法：

```sh
# help commander
npm init help
npm create help
yarn create help
# thing commander
npm init thing
npm create thing
yarn create thing
```

# 项目编辑

在此编辑项目文件并且放入目录 /src/projects.json
